# Aria Landing Page Template

A Next.js landing page template for [Aria](https://ariaagi.com). Customize branding, copy, and links in one place, then deploy.

## Customize

Edit `src/content/site-copy.ts` — site name, hero, CTAs, links (`links.app` for Get Started), logo (`brand.hasLogo`, `brand.logoSrc`), and section copy.
